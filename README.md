# Sentiment Analysis of NLP Data

In this project I have created a model to analyse the sentiments from textual data. I scraped 15k tweets when apple launched Apple IPhone XS Max and tried to figure out what was 
the overwhelming reaction of people tweeting about the product.
I have blogged about the same [here.] (https://gaurang.swarge.com/blog/sentiment-analysis-of-nlp-data-in-r)
